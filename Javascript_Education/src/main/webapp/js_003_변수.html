변수의 사용

변수의 선언 규칙
1.대소문자를 구분
2.영문,$,_로 시작해야함
3.$,_를 허용하고 다른 특수문자는 허용하지않음
4.키워드와 예약어는 사용하지 못함
변수 범위
1.전역 변수: window객체 포함되는 변수, 다른 함수들에서 공용으로 사용 할 수 있다
2. 지역변수 : 함수나 객체에 선언되고 실행이 종료되면 사라짐 (let,const)
변수의 선언과 저장
1.var/let 키워드를 사용해서 변수를 저장한다
2.const 키워드는 변수를 상수로 만듬
3.javascript변수 타입은 저장되는 값에 의해서 결정된다(동적변수)
javascript 타입의 종류
  • 문자(string)
  • 숫자(number)
  • 논리(boolean)
  • 널(object/null)
  • 객체(object/array)
  • 함수(function)
  • DOM탐색(Object)
